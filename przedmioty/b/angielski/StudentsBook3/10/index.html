Zadanie 4

Uwaga! Zdania należy napisać używając czasu Present Perfect Continuous.

Tworzymy go w następujący sposób:

Twierdzenie: I/you/we/they + have + been + verb + -ing.

he/she/it + has + been + verb + -ing.

Przeczenie: I/you/we/they + have not (haven't) + been + verb + -ing.

he/she/it + has not (hasn't) + been + verb + -ing.

Pytania: Have + I/you/we/they + been + verb + -ing...?

Has + he/she/it + been + verb + -ing...?

 

1. have you been waiting

2. haven't been waiting

3. 've been standing

4. 's been working

5. 's been learning

6. 's been looking after

7. has the Mona Lisa been hanging


Zadanie 5

Uwaga!

Czasu Present Perfect Continuous używamy, kiedy w zdaniu nacisk jest położony na długość trwania jakiejś czynności oraz z czasownikami, które nie są statyczne (wyrażające stany, uczucia, opinie, itp.). Natomiast czasu Present Perfect Simple używamy w zdaniach, w których ważny jest skutek danej czynności. Czasowniki statyczne występować będą tylko w czasie prostym - simple.

 

1. (przykład)

2. 've studied/been studying

3. 've listened/been listening

4. 've gone/been going

5. 've known

6. 've sat/been sitting

 

Uwaga! Pozostałą część zadania należy wykonać indywidualnie.


Zadanie 6

Uwaga! Pytania należy napisać używając czasu Present Perfect lub Present Perfect Continuous.

Tworzymy je w następujący sposób:

Present Perfect Simple:

(How long) Have + I/you/we/they + past participle (trzecia kolumna w tabeli czasowników nieregularnych lub czasownik regularny z końcówka -ed)?

(How long) Has + he/she/it + past participle (trzecia kolumna w tabeli czasowników nieregularnych lub czasownik regularny z końcówka -ed)?

 

Present Perfect Continuous:

(How long) Have + I/you/we/they + been + verb + -ing...?

(How long) Has + he/she/it + been + verb + -ing...?

 

1. (przykład)

2. How long have you studied/been studying English?

3. How long have you listened/been listening to the same music?

4. How long have you gone/been going to the same hairdresser's?

5. How long have you known your oldest friend?

6. How long have you sat/been sitting in this chair?

2. How long have you been wearing the same watch?

3. How long have you had the same bag?

4. How many foreign countries have you visited?
