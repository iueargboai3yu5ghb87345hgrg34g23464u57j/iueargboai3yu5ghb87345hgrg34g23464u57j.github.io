Zadanie 7

1. (przykład)

2. One

3. mean

4. benefit

5. drawbacks

6. main

7. Another

8. honest

 

Tłumaczenie:

1. (przykład)

2. One benefit is that - Jedną z korzyści jest ...

3. What I mean is that - Chodzi mi o to, że ...

4. Another benefit of doing voluntary work is - Inną korzyścią z pracy charytatywnej jest ...

5. I think there are drawbacks too - (...) ale uważam, że są także złe strony ...

6. One of the main disadvantages is - Jedną z największych wad jest ...

7. Another disadvantage is - Inną wadą jest ...

8. To be honest - Prawdę mówiąc ...
