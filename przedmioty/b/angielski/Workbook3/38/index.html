Zadanie 2

1. used to wear 

2. didn't use to be

3. used to be

4. Did short hair use to be

*Przypomnienie:

Czasownika "used to" używamy do opisywania zwyczajów i stanów z przeszłości, które w chwili mówienia o nich są już nieaktualne.

Forma przecząca tego czasownika to "didn't use to", a pytania tworzymy w następujący sposób:

"Did" + osoba + "use to" + dalsza część pytania.

Zadanie 3

1. would tell 

2. used to believe

3. didn't use to feel

4. would say

5. didn't use to know

Uzasadnienie odpowiedzi:

Zdania 1 i 4 opisują czynności wykonywane regularnie w przeszłości (mówiłem moim rodzicom / mówiłem, że będę w stanie "robić wszystko"), dlatego można tam użyć konstrukcji "would + czasownika".

Zdania 2, 3 i 5 opisują stany w przeszłości (wierzyłem, że firmy będą o mnie walczyć / Nie czułem, że dalsza nauka była potrzebna / Nie wiedziałem wtedy wiele o prawdziwym życiu), dlatego można użyć tam tylko konstrukcji "used to + czasownika".

 


Zadanie 4

1 a. (Przykład)

   b. ✔

2 a. ✔

3 a. ✔

4 b. ✔


Uzasadnienie odpowiedzi:

Podpunkt 1 opisuje czynność wykonywaną regularnie w przeszłości (wyjeżdżaliśmy samochodem wcześnie rano), dlatego poprawne jest też zdanie z konstrukcją "would + czasownika".

Podpunkt 2 opisuje wydarzenie, które wystąpiło tylko raz w przeszłości (Pewnego lata pojechaliśmy z Polski do Hiszpanii), dlatego poprawne jest tylko zdanie z formą czasu Past simple.

Podpunkty 3 i 4 opisują stan w przeszłości (uważałem za niesamowite / nie mieliśmy w tych czasach GPSa), dlatego poprawne są tylko te zdania z konstrukcją "used to + czasownika".


Zadanie 5

1. didn't use to be

2. would fall out

3. used to look up to

4. used to hate

5. would start

6. used to think


Uzasadnienie odpowiedzi:

1. Żeby uzupełnić zdanie podanym wyrazem, należy użyć konstrukcji, która w formie przeczącej ma czasownik posiłkowy 'didn't', czyli "used to + czasownik" w formie przeczącej.

2. Żeby uzupełnić zdanie podanym wyrazem, należy użyć konstrukcji "would + czasownik".

3. Żeby uzupełnić zdanie podanym wyrazem, należy użyć konstrukcji "used to + czasownik".

4. Wyraz podany w tym podpunkcie opisuje stan (hate - nienawidzić), dlatego należy użyć konstrukcji "used to + czasownik".

5. Żeby uzupełnić zdanie podanym wyrazem, należy użyć konstrukcji "would + czasownik".

6. Wyraz podany w tym podpunkcie opisuje stan (think - sądzić, uważać), dlatego należy użyć konstrukcji "used to + czasownik".
