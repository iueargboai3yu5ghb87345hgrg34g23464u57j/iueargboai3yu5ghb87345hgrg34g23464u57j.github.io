Zadanie 1

Uwaga! Zdania należy uzupełnić takimi formami czasownika, żeby powstały zdania w 1. lub 2. trybie warunkowym (first/second conditional).

Tworzymy je w następujący sposób:

First conditional:

If + Present Simple, will (won't) + verb (podstawowa forma czasownika).

lub

Will (Won't) + verb (podstawowa forma czasownika) + if + Present Simple.

Second conditional:

If + Past Simple, would(n't)/could(n't) + verb (podstawowa forma czasownika).

lub

Would (n't)/Could (n't) + verb (podstawowa forma czasownika) + if + Past Simple.

 

1. Would

2. could

3. don't

4. would

5. would

6. asks

7. will


Zadanie 2

Uwaga!

Zdania należy uzupełnić takimi formami czasownika, żeby powstały zdania w 1. lub 2. trybie warunkowym (first/second conditional).

Tworzymy je w następujący sposób:

First conditional:

If + Present Simple, will (won't) + verb (podstawowa forma czasownika).

lub

Will (Won't) + verb (podstawowa forma czasownika) + if + Present Simple.

Second conditional:

If + Past Simple, would(n't)/could(n't) + verb (podstawowa forma czasownika).

lub

Would (n't)/Could (n't) + verb (podstawowa forma czasownika) + if + Past Simple.

Jeśli zdanie warunkowe zaczyna się od "If", wtedy druga część zdania jest oddzielona przecinkiem.

Natomiast jeśli zdanie warunkowe zaczyna się od "Will", wtedy przecinka nie stawiamy.

 

1. would go crazy if he wasn't able to 

2. last all day if she charges

3. is late again, we'll get stuck

4. had more time, she would talk to her friends

5. wouldn't be able to go online if


Zadanie 3

Uwaga!

Zdania należy uzupełnić takimi formami czasownika, żeby powstały zdania w 1. lub 2. trybie warunkowym (first/second conditional).

Tworzymy je w następujący sposób:

First conditional, używamy mówiąc o wydarzeniach, które są możliwe do wystąpienia w przyszłości pod pewnymi warunkami:

If + Present Simple, will (won't) + verb (podstawowa forma czasownika).

lub

Will (Won't) + verb (podstawowa forma czasownika) + if + Present Simple.

Second conditional, używamy mówiąc o obecnych sytuacjach lub stanach, które nie mogą ulec zmianie lub zmiana ta jest bardzo nieprawdopodobna, lub o tym, co mogłoby teoretycznie wydarzyć się w przyszłości, ale jest mało realne:

If + Past Simple, would(n't)/could(n't) + verb (podstawowa forma czasownika).

lub

Would (n't)/Could (n't) + verb (podstawowa forma czasownika) + if + Past Simple.

 

1. would take

2. knew, would understand

3. won't become, doesn't go

4. would wear, lived

5. doesn't hurry, 'll miss

6. don't eat, 'll be

7. lend, will you ride

8. looked, would get tired
