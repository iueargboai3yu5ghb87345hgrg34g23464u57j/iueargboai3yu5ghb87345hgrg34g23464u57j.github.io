Uzasadnienie odpowiedzi:

1. "Wyobraź sobie, że w swoim ogrodzie zobaczyłeś UFO, co byś zrobił?" - Żeby w logiczny sposób uzupełnić to zdanie, należy użyć wyrazu wprowadzającego warunek bezpośrednio dotyczący rozmówcy (imagine).

2. "Gdyby nie byli kompetentni, nie zatrudniliby cię." - to zdanie opisuje ciągle aktualny warunek, dlatego należy uzupełnić konstrukcję 2. trybu warunkowego (If + Past Simple). Zamiast 'if' można użyć inwersji, czyli brakującego czasownika w formie Past Simple (Were).

3. "Nie zrobiłbym tego, chyba że wiedziałbym, że warto spróbować." - Żeby w logiczny sposób uzupełnić to zdanie, należy użyć wyrazu wprowadzającego warunek w formie przeczącej (Unless).

4. "Gdyby Leo nie zdradził swoich planów Catherine, nie mielibyśmy pojęcia co zamierza zrobić." - Poprawna kolejność wyrazów w szyku odwróconym (inwersji) to: Czasownik + podmiot + not + reszta zdania.

5. "Z radością pomogę ci zrobić model systemu słonecznego, pod warunkiem, że przygotujesz wszystkie kawałki i części." - Żeby w logiczny sposób uzupełnić to zdanie, należy użyć wyrazu wprowadzającego warunek (provided).


Zadanie 5

Mieszane tryby warunkowe łączą założenia 2. i 3. trybu warunkowego.

Występują dwa rodzaje takich zdań:

3+2: obecne skutki wyobrażonych sytuacji z przeszłości:

If + past perfect, would + bezokolicznik bez "to".

Oraz 2+3: wpływ wyobrażonej teraźniejszej sytuacji na przeszłość:

If + past simple, would/could + have + past participle.

 

1. wouldn't stay late if her boss hadn't been angry

2. had a smartphone, she could have

3. weren't busy, I would have sent you

4. hadn't studied acting, he wouldn't be

5. would have gone to the party if we weren't


Zadanie 6

1. would

2. wouldn't know

3. if

4. would read

5. Had we


Uzasadnienie odpowiedzi:

1. 'If it wasn't free, journalists would only be able to write certain stories.'  (Gdyby prasa nie była wolna, dziennikarze mogliby pisać tylko pewne historie.) - w II zdaniu warunkowym, w części opisującej rezultat używamy formy would + bezokolicznika.

2. 'If they hadn't published it, I wouldn't know so much about the problem now.'  (Gdyby nie opublikowali artykułu, nie wiedziałbym teraz tak wiele o tym problemie.) - w mieszanym trybie warunkowym, w części opisującej rezultat występujący obecnie używamy formy would + bezokolicznika.

3. 'They wouldn't have done that if they hadn't read about it.'  (Nie zrobiliby tego, gdyby o tym nie przeczytali.) - wyraz 'if' w logiczny sposób uzupełnia to zdanie.

4. 'If they were better written, more people would read them!'  (Gdyby były lepiej napisane, więcej ludzi by je czytało!) - w II zdaniu warunkowym, w części opisującej rezultat używamy formy would + bezokolicznika.

5. 'Had we not had it over the years, many things would have been covered up from the public.'  (Gdybyśmy przez lata nie mieli wolności, wiele rzeczy zostałoby ukrytych przed ludźmi.) - poprawna kolejność wyrazów w szyku odwróconym (inwersji), to: Czasownik + podmiot + not + reszta zdania.
